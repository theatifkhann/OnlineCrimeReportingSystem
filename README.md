# Online Crime Reporting System 🚔

## Features
- User Authentication (JWT)
- Email Verification (OTP)
- Forgot Password System
- Role-based Access (Admin/User)
- Crime Reporting System

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas

## Setup Instructions

### Backend
cd crime-backend
npm install
nodemon run dev

### Frontend
cd crime-frontend
cd citizon-portal
npm install
npm run dev
